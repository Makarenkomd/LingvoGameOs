Для разработчиков

Web-Mobile приложения

Web-Mobile приложения — это игры и образовательные приложения, разработанные с использованием веб-технологий (HTML5, CSS, JavaScript), которые оптимизированы для работы на мобильных устройствах через браузер. Они обеспечивают кроссплатформенную доступность и не требуют установки.

1. Подготовка приложения

Создайте приложение, используя HTML5, CSS, и JavaScript. Убедитесь, что оно адаптивно и работает на устройствах с разными размерами экрана (от 320px до 768px). Используйте фреймворки, такие как Phaser для игр или React для интерактивных приложений.

2. Тестирование

Проверьте приложение на популярных мобильных браузерах (Chrome, Safari, Firefox). Убедитесь, что оно соответствует стандартам WCAG 2.1 для доступности. Проверьте производительность с помощью инструментов, таких как Lighthouse.

3. Подготовка метаданных

Создайте привлекательную обложку (рекомендуемый размер: 294x424px), подготовьте скриншоты и описание. Укажите категории (например, "Словарный запас", "Грамматика") и ключевые слова для поиска.

4. Публикация

Загрузите приложение через форму на странице Загрузка игры. Укажите URL игры или загрузите ZIP-архив. После проверки модератором приложение станет доступным в разделе "Рекомендуемые игры".

Web-Desktop приложения

Web-Desktop приложения — это веб-приложения, оптимизированные для работы в браузерах на настольных компьютерах. Они подходят для более сложных образовательных игр, требующих большего экрана и производительности.

1. Подготовка приложения

Разработайте приложение с использованием HTML5, CSS, и JavaScript. Рекомендуется использовать canvas или WebGL для сложных визуальных эффектов. Убедитесь, что интерфейс удобен для работы с мышью и клавиатурой.

2. Тестирование

Проверьте приложение в Chrome, Firefox, и Edge. Оптимизируйте производительность для экранов с разрешением от 1280x720px. Используйте инструменты, такие как Chrome DevTools, для отладки.

3. Подготовка метаданных

Подготовьте обложку (294x424px), скриншоты, и подробное описание. Укажите уровень сложности (например, "Юный нарт" или "Кавказский орёл") и категории.

4. Публикация

Загрузите приложение через форму на странице Загрузка игры. Предоставьте URL или ZIP-архив. После проверки приложение появится в каталоге.

Desktop приложения

Desktop приложения — это автономные программы для Windows, macOS или Linux, которые пользователи загружают и устанавливают. Они подходят для сложных образовательных приложений с высокой интерактивностью.

1. Подготовка приложения

Разработайте приложение, используя подходящий язык (например, C++, Python с Pygame, или Unity). Убедитесь, что оно поддерживает популярные ОС (Windows, macOS, Linux).

2. Тестирование

Проверьте приложение на целевых платформах. Убедитесь, что оно не содержит ошибок и соответствует требованиям безопасности. Проверьте установочный файл (EXE, DMG, или AppImage).

3. Подготовка метаданных

Создайте обложку (294x424px), скриншоты, и описание. Укажите системные требования и поддерживаемые ОС.

4. Публикация

Загрузите установочный файл (ZIP, RAR, APK, EX выступает, DMG) через форму на странице Загрузка игры. После проверки приложение станет доступным для скачивания.
